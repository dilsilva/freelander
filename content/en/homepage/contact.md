##### How It Works

Our contact form uses GitHub Issues for submissions, providing:
- **No backend required** - Static site friendly
- **Automatic notifications** - Get emailed when someone contacts you
- **Issue tracking** - Never lose a message
- **Anti-spam protection** - GitHub account required

Simply update the `github_username` and `github_repo` in `hugo.toml` to connect the form to your repository.
